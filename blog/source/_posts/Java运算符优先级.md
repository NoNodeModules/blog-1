java运算符的优先级（1级最高）
<!--more-->
|优先级  |运算符  |
|--|--|
| 1 | ` .  () ` 点和括号  |
| 2|  `++ --` 自增自减 |
| 3 | `new ` 新建对象 |
| 4 |  `* / %` 乘除模 |
 | 5 |  `+ -` 加减 |
| 6| `>> << >>>` 右移 左移 无符号右移 |
 | 7 | `> < >= <=` 比较运算符 |
| 8 |  `== !=`  |
 | 9 |  `&` 位和 |
| 10 |  `^` 位异或 |
 | 11 |  `|` 位或 |
| 12 | `!` 逻辑非 |
 | 13 |  `&&` 短路和 |
| 14 |  `||` 短路或 |
 | 15 | `?:` 三目运算符 |
| 16 |  `= += -= *= /= %= ^=` 赋值 |
 | 17 | `&= <<= >>=`  位运算赋值 |
